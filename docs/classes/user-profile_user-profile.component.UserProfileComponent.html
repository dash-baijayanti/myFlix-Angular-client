Class UserProfileComponent

Component

Name

UserProfileComponent

Description

This component handles user profile functionalities, such as displaying user details, editing profile information, navigating to other pages (Movies or Welcome), and deleting the user account. It uses Angular services to communicate with the backend API and display relevant information or errors to the user.
